# /g/:id 참여 위저드 UI 전면 개선 (2차)

**날짜**: 2026-07-12 · 사용자 화면별 지적사항을 스펙으로 확정 (문구는 지시 원문 그대로)

## 공통 — 헤더 subtitle 3상태

카운트는 항상 `{total}명 중 {joined}명 등록` 형식 (참여 후엔 서버 최신값).

| 상태 | subtitle |
|---|---|
| 미로그인(entry) | `{total}명 중 {joined}명 등록 · 아직 {total-joined}명이 미등록 중입니다.` |
| 로그인·미참여 (위저드 진행) | `… · 당신의 자리가 비어 있어요` |
| 참여 완료 | `… · 내 관상은 이미 등록했습니다.` |

## 1. entry

- 초대장 칩: **등록(✓) 완료자 먼저**, 대기자 나중.
- 카피: `앱 설치 없이 브라우저에서 관상을 등록할 수 있습니다.`
- 카카오 버튼 폭 확대: `width:100%; max-width:360px`.

## 2. already (이미 참여)

- 제목: `등록 완료` ("이미 '…'(으)로 참여한 그룹이에요" 폐기)
- 아바타 56px (로스터와 동일 지름) + 닉네임, **우측에 `관상 다시 촬영` 버튼** (가로 배치).
- 안내: `나머지 {n}명이 등록을 마치면 그룹 케미 결과표가 공개됩니다.`
  (n=0 이면 `전원 등록 완료! 그룹 케미 결과표가 곧 공개됩니다.`)

## 3. info (정보 확인)

- 제목: `정보 확인` (앱 InfoConfirm 과 동일 명칭)
- 필드 3개 **모두 selectbox**, 일관 폼 (라벨 위 + select 아래, 폭 280 단일 컬럼, 좌정렬):
  - 인종: 동아시아인(eastAsian)/백인(caucasian)/아프리카인(african)/히스패닉(hispanic)/중동인(middleEastern) — default 동아시아인
  - 성별: 남성(male)/여성(female) — default 남성
  - 나이대: 10대~70대+ — default 20대
- 전부 default 보유 → 검증 문구 불필요, 즉시 진행 가능.
- localStorage prefill 에 ethnicity 포함.
- `카메라 켜기` 버튼: **흰 바탕 + 검정 글자 + 검정 border** (join-btn--line), 폼과 같은 폭.

## 4. camera

- **영역 사전 확보**: wrap 에 `aspect-ratio: 4/3` + max-width 320 고정 — 스트림 도착
  전에도 같은 크기의 흰 카드가 자리를 지켜 점핑 0. video 는 absolute fill + cover.

## 5. done (로스터)

- **미등록 멤버도 빈 슬롯**(점선 원 + 이름 hint 색)으로 표시 — 앱 팀룸과 동일 문법.
- 안내: `나머지 {n}명이 등록을 마치면 그룹 케미 결과표가 공개됩니다. 측면까지 넣은
  정밀 분석은 앱에서만 가능합니다.` (n=0 이면 전원 등록 완료 문구)

## 구현 포인트

- `fetchRoster` → 전 멤버 `{name, joined, thumbnailKey}` 반환으로 확장.
- `onJoined` → `{joined, total}` 전달 (헤더 카운트 최신화).
- body.ethnicity 하드코딩 제거 — 사용자 선택값 저장.
- 검증: typecheck 신규 0 · 실기기 흐름 확인 (형).
