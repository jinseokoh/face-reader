# PIVOT — 관상 소셜 (심플판, 화면 중심 한 장)

**2026-06-10** · 관상 앱에 **공개 게시판 + 명예의 전당 + 닮은꼴 콘텐츠**를 얹는다.
기존 기능(내 관상·1코인 궁합·코인)을 최대한 그대로 재활용. 이 문서는 *구현이 아니라 화면*을 러프하게 제안.

---

## 한 장 모델

```
[내 관상 보기] 무료(현재)
[궁합 보기] 내 사진 ×× 지정/앨범 사진 = 1🪙(현재)
        │
        ▼  새로 추가
[공개 게시판] 내 관상 올리기 — 썸네일 공개=무료 / 가림=유료
        │  남의 공개 관상 탐색 → [궁합 보기 1🪙]
        ▼
점수 높으면(≥금슬상화) → [명예의 전당] 자동 입성 · 7일 유지
        │  읽기 무료(공개 구경) · 글쓰기 유료(둘만)
        ▼  7일 후 전당서 내려가도 두 사람 사적 보드는 영구

[닮은꼴 ≥80] 전혀 다른 퍼널 — 별도 메뉴, 연결 아님, 그냥 재밌는 콘텐츠
```

**핵심**: 궁합은 **탭할 때만 1코인으로 계산**(온디맨드) → 미리 계산·추천 엔진 불필요. 닮은꼴만 가벼운 벡터.

---

## 코인 (제안)

| 행위 | 비용 |
|---|---|
| 내 관상 보기 · 게시판 읽기 · 전당 읽기 · 썸네일 **공개** 올리기 | **무료** |
| **궁합 보기** (지정 사진 / 게시판) | **1🪙** (현재) |
| 게시판 올리기 — 썸네일 **가림** | **1🪙** |
| 전당/사적 보드 **글쓰기** | **1🪙 / 글** |

> 보는 건 무료, 쓰는 건 유료. 얼굴 공개는 무료, 가리면 유료.

## 탭 (제안)

```
[ 관상 ]   [ 게시판 ]   [ 명예의 전당 ]   [ 닮은꼴 ]   (⚙ 설정)

 +올리기     +궁합 1🪙     공개 쇼케이스      콘텐츠
```

## 무엇이 단순해졌나

- **궁합 추천 엔진(compat_vec·MIPS·distill·배치) 불필요** — 궁합은 게시판에서 **탭당 1코인 온디맨드**.
- **bored 체크 + 필수 intro form 제거** — 게시판 올리기가 곧 참여. 프로필은 글의 한 줄 소개(선택).
- **개인 궁합 TOP 없음** — 좋은 궁합은 게시판 탐색 중 우연히 터지고, 터지면 전당이 공개.
- **닮은꼴은 연결에서 분리** — `shape_vec`(pgvector) 위 가벼운 콘텐츠 메뉴.

## 로드맵 (러프)

- **P1** 게시판(올리기 공개무료/가림유료 · 탐색 · 궁합 1🪙) — 기존 엔진 재활용
- **P2** 명예의 전당(자동 입성·7일·공개 읽기) + 전당/사적 보드 글쓰기 1🪙
- **P3** 닮은꼴 콘텐츠 메뉴(≥80 + 동물상/연예인/월드컵)
- **P4** 신고·차단 등 운영 + 코인 가격 튜닝

> 법적 검토(동의·연령 등)는 후순위로 분리.
