# PIVOT — 관상 소셜 매칭 앱 (화면 중심 한 장)

**2026-06-09** · 관상 "리포트" 앱 → 풀에서 **닮은꼴·궁합으로 사람을 연결**하는 소셜 앱.
관상은 목적이 아니라 매칭의 연료. 이 문서는 *구현 난이도가 아니라 화면 구성*을 러프하게 제안한다.
엔진 수학·DB는 `react/docs/TO-DO.md`, `flutter/docs/HOW-IT-WORKS.md` 참조.

---

## 핵심 규칙 (확정)

- **등록·공유 = 내 얼굴만** (`is_my_face`). 그 외 관상은 로컬 재미용. 공유 시도 → `WTF?! 내 얼굴만 공유돼.`
- **내 얼굴 = 카메라만** (앨범 불가). 재확인 시 **카메라 재촬영 → 얼굴 벡터 비교**로 동일인 검증(가짜 차단).
- **풀 가입 = "feeling kinda bored" 체크** → 동의 → **intro form**(직업군·관심사·MBTI·**찾는 관계**) 필수.
- **두 연결 레인**
  - **닮은꼴** = 누구나 보는 **저비용 일상 퍼널** (개인 TOP 리스트).
  - **궁합** = 개인 뷰 없음. 시스템이 **희소한 좋은 짝**(천작지합·금슬상화)을 찾으면 **"방"을 붙여 명예의 전당에 공개**.
- **매체 = 코인 글쓰기 보드** (실시간 채팅 없음). **보는 건 무료, 쓰는 건 유료.**
- **알림 없음** — 전부 pull(메뉴에서 본다). 궁합 탐지는 배치로 충분.

## 코인 (제안)

| 행위 | 비용 | 의도 |
|---|---|---|
| 풀 가입 · 리스트 열람 · **모든 읽기** | **무료** | 콘텐츠 소비·FOMO 최대화 |
| **닮은꼴 보드 글쓰기** | **1 🪙 / 글** | 대량·저마찰 일상 퍼널 |
| **명예의 전당 글쓰기** | **5 🪙 / 글** | 희소·프리미엄·prestige |
| 충전 / 광고 3편=1🪙 | — | 기존 `coins`·`ad_rewards` 재활용 |

> 쓰기에만 과금 → 진정성(돈 건 사람만 말 검)·스팸 억제·수익을 한 번에. 읽기 무료 → 전당이 공개 구경거리.

## 탭 구성 (하단 4탭)

```
[ 관상 ]   [ 닮은꼴 ]   [ 명예의 전당 ]   [ 설정 ]
 분석·내    내 닮은꼴     희소 궁합 짝       코인/지갑
 리포트     TOP+보드      공개 쇼케이스       ·계정
```

## 두 레인 요약

| | 닮은꼴 | 궁합 |
|---|---|---|
| 발견 | 개인 TOP 리스트 (pull) | 개인 뷰 **없음** — 전당에 자동 공개 |
| 누가 | 누구나 (대량) | 희소한 좋은 짝만 |
| 보드 | 1:1 사적 | 전당 방 (공개 구경) |
| 쓰기 | 1🪙 | 5🪙 |
| 읽기 | 무료 | 무료 |

## 로드맵 (러프)

- **P1** 내 얼굴만 등록(카메라+벡터검증) · bored 가입 · intro form · **닮은꼴 TOP**
- **P2** 닮은꼴 1:1 글쓰기 보드(읽기 무료/쓰기 1🪙) · 신고·차단
- **P3** 궁합 배치 선정 · **명예의 전당**(블러·대화여부·공개 읽기) · 전당 글쓰기 5🪙
- **P4** 풀 급성장 시 추천 가속(embedding) · 코인 가격 튜닝

> 법적 검토(동의·연령 등)는 후순위로 분리.
